ORIENTAÇÃO DO SISTEMA

Como o DecMET ajuda na leitura do reporte

1. Cole o METAR

Use a página Decodificador para inserir o reporte bruto exatamente como recebido.

2. Confira as partes

O sistema separa o código em grupos de vento, visibilidade, tempo, nuvens, temperatura e pressão.

3. Leia a tradução

Cada grupo recebe uma descrição em linguagem direta para estudo e consulta operacional.

4. Revise alertas

Tokens desconhecidos ou suspeitos são destacados para revisão manual do reporte.

Uso recomendado

Siga o fluxo ideal do sistema em três passos e use cada página no seu propósito: 

DecMET
├─ Aeródromos
│  └─ Busque o nome do aeródromo e copie/registre o código ICAO
├─ Consulta METAR
│  └─ Use a página para obter o METAR mais recente via NOAA (National Oceanic and Atmospheric Administration)
└─ Decodificador
   ├─ Cole o METAR bruto recebido
   ├─ Verifique vento, visibilidade, nuvens, temperatura, pressão e tendências
   └─ Revise alertas e tokens não reconhecidos para validação final

Ferramenta de aviação para METAR, ICAO e meteorologia aeronáutica

Decodificar METAR

O DecMET interpreta grupos de vento, visibilidade, tempo presente, nuvens, temperatura, ponto de orvalho e pressão, ajudando pilotos, alunos e entusiastas a entenderem o reporte bruto.

Buscar ICAO

A busca de aeródromos facilita encontrar o código ICAO correto antes da consulta METAR, reduzindo erros de identificação de estação ou aeroporto.

Consultar METAR

A consulta em tempo real centraliza o METAR bruto retornado pela API NOAA e prepara a mensagem para leitura manual ou decodificação no próprio sistema.
